# DashDrive Admin Panel - Functional Catalog

This document provides a comprehensive list of all operational modules and technical functions available in the DashDrive Admin Panel.

---

## 🛰️ 1. Market Intelligence & Optimization (Mission Control)
The brain of the platform, powered by real-time WebSocket streams and spatial AI.

| Function | Description | Key Capabilities |
| :--- | :--- | :--- |
| **Market Intelligence Hub** | Centralized Heatmap & Tactical Command. | Live Demand/Supply H3 Grid, Global Marketplace Explorer, Split-View Service Comparison (Rides vs. Food). |
| **Market Analytics** | Performance monitoring via time-series trends. | Authoritative network history snapshots, Demand vs. Supply area charts, CTA-to-Map drill-down. |
| **Traffic Insights** | City-wide road performance monitoring. | AI-driven routing optimization suggestions and congestion heatmaps. |
| **Routing Optimization** | Technical hub for pathfinding logic. | Configuration of navigation providers and cost-weighted routing paths. |

---

## 🚗 2. Operations & Live Monitoring
Real-time tracking and tactical intervention tools.

| Function | Description | Key Capabilities |
| :--- | :--- | :--- |
| **Operations Hub** | The primary situational awareness dashboard. | Marker Clustering (Google Maps), Live Driver status tracking, Active Trip monitoring. |
| **Live Tracking** | High-precision entity monitoring. | 1:1 GPS tracking for specific trips or high-value deliveries. |
| **Dispatch Management** | Fulfillment orchestration. | Auto-assignment logic, manual dispatch overrides, and ETA management. |
| **Zone Setup** | Spatial governance (Geofencing). | Polygon-based operational area definition, restricted zone management. |
| **School Run Monitoring** | Specialized niche monitoring. | Parent notifications, specific vehicle tracking for school transport services. |

---

## 📦 3. Service Verticals (Vertical Hubs)
Management consoles for each DashDrive business line.

*   **Ride Hailing**: Manage pricing, capacity, and active driver pools for standard mobility.
*   **Food Delivery**: Vendor catalogs, outlet monitoring, and delivery radius control.
*   **Mart Delivery**: Grocery and retail logistics management.
*   **Parcel Delivery**: Courier assignment and last-mile logistics tracking.
*   **Shopping**: Concierge service management for personal shoppers.
*   **City to City**: Long-distance hub-to-hub trip management.
*   **Car Rental**: Reservation system for self-drive or chauffeur-driven rentals.

---

## 💰 4. Finance, Fintech & Pricing
The economic engine of the DashDrive ecosystem.

| Function | Description | Key Capabilities |
| :--- | :--- | :--- |
| **Financial Reports Hub** | Deep-dive accounting and treasury. | Earnings, Expenses, Cash flow, and Tax compliance dashboard. |
| **Fare Management** | Dynamic pricing engine. | Base fare setup, distance/time pricing, and surge multiplier logic. |
| **Fintech Partner Hub** | 3rd party financial integrations. | BNPL (Buy Now Pay Later), specialized insurance, and utility payments. |
| **Settlements** | Payout automation. | Automated driver/merchant payout processing and wallet reconciliations. |
| **Marketplace Management** | High-level SKU governance. | Service-wide commission settings and marketplace fee configurations. |

---

## 👥 5. Partners, Fleet & Users
Registry and lifecycle management for all platform participants.

*   **Driver Management Hub**: Registry for all drivers with multi-stage verification (KYC/KYB).
*   **Fleet Operator Hub**: Dedicated interface for external fleet companies.
*   **Vehicle Management**: VIN tracking, vehicle attribute mapping (AC, Luxury), and maintenance logs.
*   **Customer & User Management**: User profiles, interaction history, and segmenting.
*   **Driver Rewards & Retention**: Loyalty tiers and performance-based incentive engines.

---

## 📣 6. Growth, Marketing & Content
User acquisition and engagement tools.

*   **Marketing Hub**: Centralized campaign management for coupons and promos.
*   **Growth Engine**: Referral tracking and viral growth loop analytics.
*   **App CMS (Content Manager)**: Manage mobile home screen banners, news carousels, and landing pages.
*   **Create Blog**: Internal content platform for regional updates and driver education.

---

## 🛡️ 7. Security, Trust & Technical Support
High-integrity governance and crisis management.

| Function | Description | Key Capabilities |
| :--- | :--- | :--- |
| **Fraud Management** | Protection against platform abuse. | Detection of phantom trips, payment fraud, and account sharing. |
| **Audit Logs** | Accountability tracking. | Record of every action taken by every Admin/Operator. |
| **Support Hub** | Multi-channel communication. | Ticket management, Live Chat, and Driver/Customer dispute resolution. |
| **Technical Resolution** | L2/L3 console for engineers. | Direct system error logs and low-level debugging console. |

---

## ⚙️ 8. Enterprise & System Configuration
Global environment and governance settings.

*   **Enterprise Setup**: High-level organizational hierarchy and regional branch management.
*   **System Settings**: Global environment variables, API keys, and maintenance mode toggles.
*   **API Management**: Monitoring of Google Maps, Weather, and Payment gateway usage.
*   **Marketplace Config**: Deep configuration of platform-wide logic and operational hours.
